# 開発言語
- Ruby 3.0.1
- Ruby on Rails 6.1.7

# 就職Termの技術
- devise
- Ajaxコメント機能
- AWS

# カリキュラム外の技術
- ransack
- カレンダー機能

# 実行手順
```
$ git clone git@github.com:harufumi03/Cucina_app.git
$ cd cucina_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
